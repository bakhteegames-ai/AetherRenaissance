## 1. PROJECT IDENTITY & PHILOSOPHY
**Genre:** Competitive RTS (Starcraft/Warcraft 3 hybrid).
**Setting:** **Aetherpunk / Clockwork Fantasy**.
* **KEY VISUAL RULE:** No "Magic" in the traditional sense. Everything is technology: gears, steam, optics, biology, or cosmic energy.
* **Philosophy:** "Decisions > Reactions". Macro-oriented. High lethality, clear counters.
* **Esports Naming Convention:** All units have 1-2 syllable "Callout Names" for fast commentary (e.g., "Volt" instead of "Volt-Arbalest").

## 3. CORE TECHNICAL RULES (FROM TDD v1.5.1)

### Economy
* **Solaris:** Main resource (Gold).
* **Quartz:** Tech resource (Wood/Gas). Required for T2/T3 and **Overcharge**.
* **Overcharge:** Ability to speed up a building by spending Quartz.
    * *Constraint:* Cannot bank the buff. Timer ticks even if production is paused.

### Physics & Movement
* **No Push:** Units do not push each other physically.
* **Smart Yield:** Idle units step aside for moving units (Vector calculation: Center-to-Center).
* **Anti-Stall:** Sudden Death triggers if Player has 0 Buildings AND 0 Active Construction Sites (must be building for S seconds).

### Damage Types (RPS)
* **Kinetic** > Light Armor (Workers, T1 Ranged).
* **Aether** > Medium Armor (T1 Melee, Casters).
* **Armor-Piercing** > Heavy Armor (Mechs, Tanks).
* **Siege** > Fortified (Buildings). *Has Friendly Fire.*

## 4. CONTEXT FOR AI GENERATION (META-INSTRUCTIONS)

* **When generating Code:** Use the Callout Names for class names (e.g., `Unit_Servo`, `Unit_Glitch`) but include the full descriptive name in comments. Adhere strictly to the interfaces defined in TDD v1.5.1.
* **When generating Art/Visuals:** Strictly avoid "High Fantasy" tropes. No wizards with wooden staffs (unless it's a cyborg-druid). No dragons (unless it's a robot). Focus on materials: Brass, Rust, Wood, Void.
* **When generating Voice Lines:**
    * *Concordat:* Disciplined, Latin-sounding, Military radio filter.
    * *Ferrum:* Grunting, loud machinery noise, shouting over engine roar.
    * *Coil:* Whispering, hive-mind distortion, squishing sounds.
    * *Entropy:* Reverse reverb, metallic scraping, glitchy/static voice.
