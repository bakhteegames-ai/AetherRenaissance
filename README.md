# Aether Renaissance

A real-time strategy (RTS) game inspired by Warcraft 3, featuring strategic resource management, hero development, and tactical combat.

## Overview

Aether Renaissance is an RTS game that combines classic RTS mechanics with innovative resource and hero management systems. Players must balance economic development, military strategy, and hero progression to achieve victory.

## Key Features

### Core Gameplay
- **Resource Management**: Three-tier resource system (Crystal, Biomass, Aether)
- **Hero System**: Hero development with experience, talents, and equipment
- **Unit Control**: Traditional RTS unit control with formations and tactical abilities
- **Base Building**: Strategic placement of structures and defenses

### Game Mechanics
- **Upkeep System**: Dynamic resource penalty based on army size
- **Fog of War**: Strategic map vision system
- **Day/Night Cycle**: Affects unit visibility and abilities
- **Creep Camps**: Neutral units for experience and rewards

### Victory Conditions
- Destruction of enemy main structure
- Control of key map objectives
- Economic domination
